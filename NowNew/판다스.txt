concat()으로 행 추가하기
concat()은 주로 여러 데이터프레임이나 시리즈 등을 결합할 때 사용하는 함수로, 데이터프레임에 단일 값이나 값의 리스트를 직접 concat()으로 추가하는 것은 일반적으로 지원되지 않습니다. 
concat()을 이용해 데이터를 추가하면 기존 데이터프레임은 변경하지 않고 데이터가 추가된 새로운 데이터프레임을 반환합니다. 따라서 기존 데이터프레임에 concat()을 통해 추가한 내용을 반영하고자 할 때는, concat()이 반환한 새로운 데이터프레임을 기존 데이터프레임 변수에 재할당해야 합니다.
concat() 함수의 사용법은 pd.concat([데이터프레임명, 새로운 행 데이터], ignore_index=True)로 "새로운 행 데이터"의 위치에 시리즈 객체나 딕셔너리, 새 데이터프레임을 넣어줍니다.

# 시리즈 객체 추가
new_series = pd.Series([9, '이상순', 40000, 3000], index=['주문번호', '고객명', '주문금액', '배송료'])
df = pd.concat([df, new_series.to_frame().T], ignore_index=True)

# 딕셔너리 추가
new_dict = {'주문번호': 10, '고객명': '이효리', '주문금액': 35000, '배송료': 3000, '배송상태': '배송중'}
df = pd.concat([df, pd.DataFrame([new_dict])], ignore_index=True)

# 새로운 주문 정보를 포함한 데이터프레임 생성
new_orders_df = pd.DataFrame({'주문번호': [11, 12],
                '고객명': ['김종국', '하하'],
                '주문금액': [25000, 15000],
                '배송료': [3000, 3000]})

# 결과 출력

concat 함수로 데이터를 결합할 때 열 방향으로 결합도 가능합니다. 열로 데이터를 추가하기 위해서는 매개변수 axis의 값을 1로 지정하면 됩니다.

# 결합할 시리즈 생성
concat_series = pd.Series(

# 데이터프레임과 시리즈를 열 방향으로 결합

행/열 삭제하기
drop() 함수를 사용합니다. 데이터프레임명.drop('행 인덱스 or 열 이름', axis=0 or 1)의 형식으로 행이나 열 삭제를 실행합니다.
단일 행 또는 단일 열을 삭제할 때는 해당 행 인덱스/열 이름을 그대로 전달하고, 여러 행/열을 삭제할 때는 삭제할 행/열 리스트를 전달합니다.
axis는 어떤 방향으로 삭제를 진행할지, 이 옵션을 별도로 지정하지 않거나 axis=0을 설정해주면 행 기준으로 작업을 하고, axis=1로 설정하면 열 기준으로 삭제 작업을 진행하게 됩니다.
inplace=True로 설정하면 원본 데이터프레임에서 삭제를 직접 진행합니다. 즉, 기존 데이터프레임이 바로 변경됩니다. 기본값은 inplace=False로 inplace 옵션을 별도로 지정하지 않으면 기존 데이터프레임을 변경하지 않고 행/열이 삭제된 새로운 데이터프레임을 반환합니다.

데이터프레임을 다시 할당하는 방법
특정 행을 선택하는 방법이나 슬라이싱, 특정 조건으로 데이터 선택하는 방법들
삭제해야하는 행이나 열이 많거나 남기고자 하는 데이터가 적을 때는 이 방법이 더 효율적일 수 있습니다.
데이터프레임 df에서 '주문번호', '고객명' 열을 제외한 모든 열을 삭제하고자 한다면 df = df[['주문번호', '고객명']]

3) 결측치 처리
결측치(null 또는 NaN)란 데이터셋에서 값이 누락되거나 제공되지 않아 값이 없는 항목을 의미

isnull()
isnull()은 데이터프레임이나 시리즈 내에서 결측치를 찾는 데 사용되는 함수
이 함수는 데이터의 각 요소에 대해 결측치 여부를 검사하고, 해당 요소가 결측치라면 True, 아니라면 False를 반환하는 불(Bool) 값으로 이루어진 데이터프레임 또는 시리즈를 생성합니다.
sum() 함수와 함께 사용하여 각 열이나 행에 있는 결측치의 수를 계산할 수 있습니다. 
# 각 열의 결측치의 개수 확인
missing_values_count = df.isnull().sum()
