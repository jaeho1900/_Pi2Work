fillna()
fillna() 함수는 데이터프레임이나 시리즈의 결측치를 다른 값으로 채우는 데 사용
.fillna(value)를 붙여서 사용하며 이 값은 스칼라 값 일 수도 있고, 딕셔너리, 시리즈 또는 데이터프레임일 수도 있습니다.
df.fillna('확인중')
df['배송상태'].fillna('배송중')
df['배송지역'].fillna(df['배송지역'].mode()[0], inplace=True)    # mode()함수는 해당 시리즈의 최빈값을 시리즈로 반환

ffill(), bfill()
ffill()은 결측치보다 이전에 있는 값으로 결측치를 채우고, bfill()은 결측치의 뒤에 있는 값을 가져와 결측치를 채우는 메서드입니다.
df.ffill()

dropna()
dropna()는 데이터프레임이나 시리즈에서 결측치를 포함하는 행이나 열을 제거할 때 사용
axis: 결측치를 제거할 축을 지정합니다. axis=0 (또는 axis='index')는 행을 제거하고, axis=1 (또는 axis='columns')은 열을 제거합니다. 기본값은 0입니다.
how: 결측치가 있는 행 또는 열을 어떻게 제거할지를 지정합니다. how='any'는 결측치가 하나라도 있는 행 또는 열을 제거하고, how='all'은 모든 값이 결측치인 행 또는 열만 제거합니다. 기본값은 how='any'입니다.
thresh: 행 또는 열에서 결측치가 아닌 값이 thresh로 설정한 값보다 많은 경우에만 해당 행 또는 열을 유지합니다. 예를 들어, 'thresh=2'라면 결측치가 아닌 값이 2개 이상 있는 행만 유지한다는 의미입니다.
subset: subset 매개변수에 열 이름의 리스트를 전달하면, 오직 이 리스트에 포함된 열들에서만 결측치를 찾습니다. 지정된 열들 중 하나라도 결측치를 포함하고 있는 행 전체가 제거됩니다. axis=0일 때만 유효합니다. 예를 들어, 'subset=['A', 'B']'를 전달하면 'A'와 'B' 열에 대해서만 결측치를 확인하고, 해당 조건을 만족하는 행을 제거합니다.
inplace: True로 설정하면, 원본 데이터프레임을 직접 수정합니다. 기본값은 False이며, 이 경우 수정된 새 데이터프레임이 반환됩니다.
df.dropna(axis=1)


4) 정렬

sort_values()
sort_values() 함수는 데이터프레임의 값을 기준으로 행이나 열을 정렬
데이터프레임명.sort_values(by='행 인덱스/열 이름' 또는 ['행 인덱스/열 이름 리스트'])의 형태로 사용
axis: 정렬을 수행할 축을 지정합니다. 0은 행을, 1은 열을 의미합니다. 기본값은 0으로 axis를 별도로 지정하지 않으면 열 값을 기준으로 행을 정렬합니다.
ascending: 정렬 순서를 오름차순(True) 또는 내림차순(False)으로 지정합니다. 기본값은 True로, 값이 작은 것에서 큰 것 순서로 정렬됩니다.
inplace: True로 설정하면, 원본 데이터프레임을 직접 수정합니다. 기본값은 False이며, 이 경우 수정된 새 데이터프레임이 반환됩니다.
na_position: 결측치의 위치를 지정합니다. 'first'는 결측치를 앞에, 'last'는 결측치를 뒤에 배치합니다.
df.sort_values(by=['열1', '열2'], ascending=[True, False])

sort_index()
sort_index() 함수는 데이터프레임의 인덱스를 기준으로 행 또는 열을 정렬
df.sort_index(axis=1)


5) 인덱스 설정 및 재설정

set_index()
특정 열을 인덱스로 설정할 수 있습니다. 데이터프레임명.set_index('열 이름')의 형식으로 실행하며 추가적으로 인덱스 설정 옵션을 지정할 수 있습니다.
drop: 특정 열을 인덱스로 설정하면서 해당 열을 그대로 데이터프레임의 열로 남겨둘지 제거할지를 지정하는 매개변수입니다. 기본값은 True로, 이 경우 인덱스로 설정한 열은 데이터프레임의 열에서 제거됩니다. False로 설정하면, 열이 인덱스로 설정되어도 데이터프레임에 그대로 남아있습니다.
append: 기존 인덱스의 유지/제거를 결정합니다. 기본값은 False로, 이 경우 기존 인덱스는 제거되고 새로운 열이 인덱스로 설정됩니다. True로 설정하면, 기존 인덱스를 유지하고 새 열을 추가 인덱스로 설정합니다.
inplace: True로 설정하면, 원본 데이터프레임을 직접 수정합니다. 기본값은 False이며, 이 경우 수정된 새 데이터프레임이 반환됩니다.
verify_integrity: 인덱스에 중복 값이 없는지 검사합니다. 중복이 있을 경우 오류를 반환합니다.
df.set_index('주문번호', inplace=True)

reset_index()
기존 인덱스를 열로 변환하고 인덱스를 재설정
level: 멀티인덱스(다중 인덱스) 데이터프레임에서 재설정할 인덱스 레벨을 지정합니다. 여러 레벨 중 하나 또는 일부를 선택하여 인덱스를 재설정할 수 있으며, level을 지정하지 않으면, 모든 레벨의 인덱스가 재설정됩니다.
drop: True로 설정하면, 기존 인덱스를 삭제하고 새로운 정수 인덱스를 생성합니다. 기본값은 False로, 이 경우 기존 인덱스는 데이터프레임의 새로운 열로 추가됩니다.
inplace: True로 설정하면, 원본 데이터프레임을 직접 수정합니다. 기본값은 False이며, 이 경우 수정된 새 데이터프레임이 반환됩니다.
df.reset_index(inplace=True)


6) 중복값 처리

duplicated()
duplicated() 메서드는 데이터프레임에서 중복된 행을 찾을 때 사용, 불리언 시리즈를 반환
특정 열에 대해 중복을 확인하기 위해서는 subset 인자를 사용해 단일 열 또는 열 리스트를 지정합니다.
df.duplicated()
df.duplicated(subset='고객명')

drop_duplicates()
drop_duplicates() 메서드는 중복된 행을 제거할 때 사용
subset 매개변수에 단일 열이나 열 리스트를 전달하여 특정 열(들)을 기준으로 중복을 제거할 수 있습니다.
df.drop_duplicates(subset='고객명', inplace=True)


7) 연산

산술 연산
기본적인 산술 연산에는 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/) 등이 있습니다. 이 연산들은 데이터프레임의 모든 원소에 적용됩니다.
df/2

판다스의 내장 함수를 이용하는 방법도 있습니다. 더하기는 add(), 빼기는 sub(), 곱하기는 mul(), 나누기는 div()를 사용
데이터프레임 간 산술 연산을 진행할 때, 판다스는 인덱스와 열 이름을 기준으로 자동으로 데이터를 정렬하고 연산을 수행,일치하는 인덱스나 열이 없을 경우 NaN 값(결측치) 할당
누락된 데이터를 처리하는 fill_value 매개변수를 사용, fill_value 매개변수는 한 데이터프레임에는 존재하지만 다른 데이터프레임에는 없는 요소에 대해 사용, 예를 들어, fill_value=0을 사용하면 누락된 값을 1로 간주하여 연산을 수행합니다.
df.add(10)
df1.mul(df2, fill_value=1)
