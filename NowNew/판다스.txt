시리즈의 인덱스, 값, 그리고 데이터 타입을 각각 분리해서 가지고 올 수도 있습니다. 시리즈 데이터에서 인덱스는 s.index로, 값은 s.values로, 마지막으로 데이터 타입은 s.dtype으로 가지고 옵니다. 위에서 만든 시리즈 데이터s에 대해 print로 각각 인덱스, 값, 데이터 타입을 출력해보겠습니다.
print(s.index)
print(s.values)
print(s.dtype)

시리즈의 인덱스를 활용하여 특정 위치의 값을 가져올 수도 있습니다.
s = pd.Series([1, 2, 3, 4], index=['a', 'b', 'c', 'd'])  # 고유 인덱스를 가진 1차원 배열 형태의 데이터 구조
print(s['c'])          # 인덱스
print(s[['b', 'c']])   # 리스트 형태
print(s[s > 2])     # 조건

특정 데이터 선택하기

열 데이터 선택하기
'열 이름'을 직접 지정하는 방식
데이터프레임명['열 이름']  # 단일 열(시리즈 반환), 단일 열 선택시에도 대괄호를 두 번 사용하면 데이터프레임으로 반환
df[['도시', '이름']]  # 복수 열(리스트에 전달, 데이터프레임으로 반환)

.iloc[] 메서드나 .loc[] 메서드를 사용
.loc[] 메서드로 열 데이터에 접근하는 것도 가능합니다. [행, 열] 형식의 인덱싱을 사용하여 특정 행과 열을 선택할 수 있습니다. 먼저 단일 열을 선택하기 위해서는 행 부분에서 모든 행(:)을 선택하고, 두 번째 인자로 열 이름을 제공합니다.
df.loc[:, ['도시', '이름']]   # 여러 행을 선택했을 때는 열의 이름을 리스트로 전달
df.loc[:, '나이':'도시']
.loc[행, 열]  # 행의 인덱스와 열 이름을 전달하여 특정 위치의 값을 선택


행 데이터 선택하기
1) .head()와 .tail()

option_context()  # 데이터프레임의 출력되는 행/열의 갯수를 원하는 양만큼 볼 수 있도록 일시적으로 설정
with 문과 함께 사용되어 해당 코드 블록 내에서만 옵션을 일시적으로 변경하는 함수
with pd.option_context('display.max_rows', 출력할 행의 수, 'display.max_columns', 출력할 열의 수):
    pd.set_option("show_dimensions", False)
    display(데이터프레임명)

2) .loc[]  # 행의 이름(인덱스)을 이용해 행 데이터를 선택

df.loc[[0, 2]]  # 리스트로 여러 행 선택
df.loc[1:2] # 연속된 여러 행을 선택할 때는 슬라이스를 .loc[] 메서드에 전달

3) .iloc[]  # 위치 기반 인덱싱을 사용하여 데이터프레임의 특정 행이나 열에 접근, 위치 기반 인덱싱이란 정수 인덱스를 의미

마지막으로 비연속적인 행이나 열을 선택할 때는 리스트를 사용합니다.

4) 슬라이싱(Slicing)
파이썬 리스트 슬라이싱과 유사하게 판다스 데이터프레임에서도 슬라이싱으로 범위를 지정하여 행 데이터를 선택할 수 있습니다.

5) 조건을 사용한 선택 
데이터프레임명[조건]의 형식으로 조건을 지정
& (AND), | ( OR), ~ (NOT) 기호를 이용
df[(df['나이'] > 20 ) & (df['도시'] != '서울' )]

불 인덱싱에 판다스의 isin() 메서드를 사용하면 지정한 리스트나 시리즈를 포함하는 행/열의 데이터를 찾아 필터링합니다
df[df['도시'].isin(['대구','부산'])]
