엑셀에서 주어진 데이터를 기반으로 분산형 차트를 만드는 방법을 안내해 드리겠습니다. 아래의 단계를 따라 진행해 보세요.

1. **데이터 입력**: 엑셀 시트에 주어진 데이터를 입력합니다. 각 열의 제목은 '상품', '정량', '정성', '평가월'로 설정합니다.

2. **차트 삽입**:
   - 데이터를 선택합니다. (A1:D25)
   - 상단 메뉴에서 '삽입' 탭을 클릭합니다.
   - '차트' 그룹에서 '분산형 차트'를 선택합니다. (일반적으로 '분산형 차트' 아이콘을 클릭하면 여러 옵션이 나타납니다.)

3. **데이터 시리즈 설정**:
   - 차트가 생성되면, 차트 영역을 클릭하여 '차트 도구'가 나타나면 '디자인' 탭을 선택합니다.
   - '데이터 선택'을 클릭합니다.
   - '데이터 원본 선택' 대화 상자가 열리면, '추가' 버튼을 클릭하여 각 상품(A, B, C, D)에 대해 별도의 데이터 시리즈를 추가합니다.
   - 각 시리즈에 대해 '정량'을 x값으로, '정성'을 y값으로 설정합니다.

4. **색상 설정**:
   - 각 데이터 시리즈를 클릭하여 색상을 변경합니다. 각 상품에 대해 다른 색상을 선택하여 구분할 수 있습니다.

5. **데이터 레이블 추가**:
   - 차트에서 데이터 포인트를 클릭하여 선택합니다.
   - 마우스 오른쪽 버튼을 클릭하고 '데이터 레이블 추가'를 선택합니다.
   - 데이터 레이블을 클릭한 후, 다시 마우스 오른쪽 버튼을 클릭하고 '데이터 레이블 서식'을 선택합니다.
   - '레이블 옵션'에서 '값에서 셀'을 선택하고 '평가월' 열의 셀 범위를 지정합니다.

6. **차트 제목 및 축 제목 추가**:
   - 차트 제목을 클릭하여 원하는 제목으로 변경합니다.
   - x축과 y축에 제목을 추가하려면, '차트 요소 추가'에서 '축 제목'을 선택하고 적절한 제목을 입력합니다.

이 과정을 통해 '정량'을 x축, '정성'을 y축으로 하여 상품 구분에 따라 색깔이 다른 분산형 차트를 만들 수 있습니다.
