함수 :  특정 작업을 수행하는 코드 블록
함수 이름 뒤에는 ()괄호로 매개변수들을 넣어줍니다. 매개변수는 파라미터(parameter)라고도 부르며, 함수에서 매개변수로 전달하는 실제 값은 인자(argument) 또는 인수라고 부릅니다.
만약 코드 실행 후에 어떤 값을 반환해야할 때는 마지막 줄에 'return 반환값'을 입력합니다. 여러 값을 반환하려면 return 뒤에 반환값을 쉼표로 나열하면 됩니다. 이렇게 반환되는 값들은 튜플로 묶여서 반환됩니다.

지역변수와 전역변수

함수를 사용하면 함수 안에서 변수를 생성하고 해당 변수로 다른 작업을 수행하게 됩니다. 이렇게 함수 안에서 생성된 변수를 지역변수(local variable)라고 합니다. 그리고 이와 반대되는 개념으로 함수 밖에서 생성된 변수는 전역변수(global variable)라고 합니다.

함수 내부에서 정의된 변수(지역 변수)는 함수를 실행하는 동안 함수 안에서만 사용할 수 있으며 함수 외부에서는 접근할 수 없습니다. 함수를 실행하는 동안만 사용되기 때문에 함수가 종료되면 지역변수는 메모리에서 제거됩니다. 반면 전역변수는 함수 안이나 밖 어디서나 사용할 수 있으며, 프로그램이 종료될 때까지 메모리에 남아있습니다.

아래의 예제를 통해 지역변수와 전역변수의 개념을 이해해보겠습니다.

x = 10  # 전역변수

def my_function():  
    y = 5   # 지역변수
    print("지역변수 y =", y)
    print("전역변수 x =", x)

my_function()
print(y)

전역변수는 함수 안에서도 사용이 가능하지만, 만약 함수 안에서 전역변수의 값을 변경하려면 global 키워드(global 전역변수명)를 사용해야합니다. 만약 global을 사용하지 않고 함수 안에서 전역변수명에 다른 값을 할당하면, 파이썬은 기존의 전역변수의 값을 변경한 것이 아닌 함수 안에서 새로운 지역변수를 생성하는 것으로 간주합니다.
x = 10
y = 5

def update_x():
    global x
    x = 20
    y = 20
    print("함수 안에서 호출한 x와 y값:", x , y)

update_x()
print("함수 밖에서 호출한 x와 y값:", x , y)
# 결과값
함수 안에서 호출한 x와 y값: 20 20
함수 밖에서 호출한 x와 y값: 20 5

클래스(Class)
함수에서 더 나아간 개념인 클래스는 일종의 설계도나 틀과 같은 것으로, 이 설계도를 토대로 "객체"를 만들어냅니다.
클래스로부터 생성된 각 객체를 클래스의 인스턴스라고 하며, 이러한 인스턴스는 고유한 속성값을 갖고 클래스에서 정의된 메서드를 공유합니다.
메서드(method)란 클래스에서 정의한 함수로, 해당 클래스의 인스턴스(객체)와 연관됩니다.

1) 클래스 선언하기
클래스명 뒤에는 소괄호()와 콜론(:)을 순차적으로 입력하고 줄바꿈을 해줍니다. 이 때 소괄호 안에는 특정 클래스나 여러 클래스를 명시하여 해당 클래스를 상속받을 수 있습니다. 특정 클래스를 상속 받지 않는 경우에는 괄호를 생략할 수 있습니다.

[기본구조]
class Car:
    # 클래스 변수 (선택적)
    total_cars = 0  # 전체 생성된 자동차의 수를 저장하는 클래스 변수
   
    # 초기화 메서드 (생성자, 선택적)
    def __init__(self, brand, model):
        # 인스턴스 변수
        self.brand = brand  # 자동차의 브랜드 초기화
        self.model = model  # 자동차의 모델명 초기화

        # 새로운 자동차 객체가 생성될 때마다 전체 자동차 수를 1씩 증가
        Car.total_cars = Car.total_cars + 1

    # 메서드
    def display_info(self):
        """자동차의 정보를 출력하는 메서드"""
        return f"이 자동차는 {self.brand} {self.model}입니다."

클래스에서 사용하는 변수에는 클래스 변수(class variable)와 인스턴스 변수(instance variable)가 있습니다. 
클래스변수는 클래스를 통해 접근(클래스명.클래스 변수명)할 수도 있고 인스턴스를 통해 접근(인스턴스명.클래스 변수명)할 수도 있습니다. 주로 해당 클래스의 모든 객체들에 공통적으로 사용되는 값을 저장할 때 사용됩니다.
인스턴스 변수는 인스턴스를 통해서만 접근(인스턴스명.인스턴스 변수명)할 수 있고, 'self'를 사용하여 정의(self.인스턴스 변수명 = 값)합니다. 이 변수는 특정 클래스 인스턴스에 속하며, 주로 각 객체의 고유한 상태나 데이터를 저장할 때 사용됩니다.

2) 객체 생성 및 활용하기
객체를 생성한 후에는 인스턴스의 변수에 접근해 속성을 가져오거나 메서드를 사용할 수 있습니다.

# 클래스를 사용하여 객체 생성
car1 = Car("현대", "소나타")
car2 = Car("기아", "K5")

# 객체의 메서드 호출
print(car1.display_info()) 
print(car2.display_info())  

# 클래스 변수에 접근
print(Car.total_cars)


3) 클래스 상속
클래스 상속은 객체 지향 프로그래밍에서의 핵심 개념 중 하나로, 어떤 클래스(부모 클래스 또는 슈퍼 클래스)의 속성과 메서드를 다른 클래스(자식 클래스 또는 서브 클래스)가 물려받을 수 있게 하는 작동 원리입니다. 상속을 사용하면 코드의 재사용성을 높일 수 있으며, 기존 코드를 변경하지 않고 기능을 확장하거나 수정할 수 있습니다. 상속을 받을 때는 class 키워드를 사용하여 클래스를 선언할 때 클래스명 뒤 소괄호() 안에 상속받을 부모 클래스명을 입력합니다.
자식 클래스는 부모 클래스의 모든 속성과 메서드를 상속받지만, 필요에 따라 일부를 재정의(overriding)하여 사용할 수 있습니다.

class 부모클래스:
    ...

class 자식클래스(부모클래스):
    ...

# 부모 클래스로 사용할 Animal 클래스
class Animal:
    def __init__(self, name):
        self.name = name

    def make_sound(self):
        print("동물의 소리")

# Animal 클래스를 상속받는 Dog 클래스
class Dog(Animal):
    def make_sound(self):  # Animal 클래스의 make_sound 메서드를 재정의
        print(f"{self.name}가 멍멍!")

# Animal 클래스를 상속받는 Cat 클래스
class Cat(Animal):
    pass  # 고양이 클래스는 Animal 클래스의 모든 메서드를 그대로 상속

# 객체 생성 및 활용
dog1 = Dog("맥스")
dog1.make_sound()

cat1 = Cat("피치")
cat1.make_sound()

모듈(module)
어떤 특정한 작업을 하기 위해 관련된 여러 변수와 함수, 클래스 등을 모아서 만들어진 하나의 .py 파일을 "모듈"이라고 합니다. 
모듈은 import 모듈명으로 불러와서 사용합니다. 
모듈에 포함된 함수나 클래스, 변수를 사용할 때는 모듈.함수(), 모듈.클래스(), 모듈.변수와 같은 형식으로 사용합니다.
또 다른 방법은 from 모듈명 import 변수명(함수명/클래스명)으로 지정된 함수나 변수만 불러오는 것입니다. 이 경우 모듈 이름을 접두사로 사용할 필요가 없어 함수명()와 같은 방식으로 간결한 코드 사용이 가능합니다. 
from 모듈명 import *를 사용하여 해당 모듈에 정의된 모든 변수, 함수, 클래스 등을 직접 가져오는 방식도 있습니다. 이 방식을 사용하면 모듈 내의 모든 항목을 모듈명 없이 바로 사용할 수 있습니다.

패키지(Package)
패키지는 여러 모듈을 묶어놓은 구조를 말합니다. 
패키지는 디렉터리와 모듈로 구성되어 있습니다. 디렉터리 안에는 여러 모듈과 서브 디렉터리(서브 패키지)가 포함될 수 있습니다.
패키지명[.하위폴더명].모듈명.변수명/함수명/클래스명의 형식으로 호출해야합니다.
파이썬 패키지 저장소(https://pypi.org/)에서 필요한 패키지를 검색하고 패키지에 대한 정보를 확인할 수 있습니다.

패키지를 불러오기 위해 import 문을 사용할 때 몇가지 주의할 점이 있습니다. 하나는 파이썬의 파이썬의 표준 라이브러리 중 하나인 sys.path에 패키지가 위치한 디렉터리가 포함되어 있어야 한다는 것입니다.
sys.path 에 경로를 추가할 때는 append를 사용합니다.
import sys
sys.path.append('/path/to/directory')

라이브러리(Library)
라이브러리는 하나 이상의 관련 모듈이나 패키지들의 집합을 의미합니다. 
"패키지"와 "라이브러리"라는 용어는 종종 혼용되어 사용되기도 합니다.

os 모듈
os.path.join() 함수는 여러 개의 경로 구성 요소를 안전하게 결합하여 하나의 경로 문자열을 생성하는 데 사용
여러 플랫폼에서 예상대로 동작
import os
path1 = "mydir"
path2 = "myfile.txt"
# os.path.join()을 사용하여 경로 결합
full_path = os.path.join(path1, path2)
print(full_path)

os.path.split() 함수는 파일 경로를 분리하여 디렉터리와 파일명으로 반환합니다.
import os
path = "C:/Users/UserName/Documents/example.txt"
directory, filename = os.path.split(path)
print("디렉터리:", directory)
print("파일명:", filename)
