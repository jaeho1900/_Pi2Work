* Repository 최신화 하기 (내려 받기)
Repository 경로에서 마우스 오른버튼 클릭 해서 "Git Bash Here"을 클릭하면
커맨드 창이 열리는데 아래 명령어 실행하면 된다

﻿git pull origin master
​
* 원격 저장소에 파일 올리기
"Git Bash Here"을 클릭해서 커맨드 창이 열리면 아래 명령어를 순서대로 실행한다

﻿git add --all
git commit -am "업로드 메시지 적기"
git push -u origin master﻿
​
* 원격 저장소의 특정 branch 를 clone 하기
git clone -b 브랜치명 --single-branch URL주소
